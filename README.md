# MeNet

A ComputerCraft program for the TerraFirmaGreg modpack that enables interplanetary Applied Energistics 2 (AE2) ME requests between separated networks. Items are sent via railgun for seamless cross-planet logistics.

## Features

- **Interplanetary ME Requests**: Request items from remote ME networks across planets.
- **Railgun Integration**: Automated item transportation using railguns.
- **User-Friendly UI**: Built with Basalt for an intuitive interface.
- **ComputerCraft Compatible**: Designed for ComputerCraft nodes in modded Minecraft.

## Usage

Run the main script to start the MeNet interface. Use the Basalt-based UI to configure and send requests between ME networks.
